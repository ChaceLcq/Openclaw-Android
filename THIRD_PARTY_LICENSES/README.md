This directory contains third-party license notices for assets and native
runtime components bundled by Openclaw-Android.

Current status:

- `MANROPE_OFL.txt` covers the bundled Manrope font.
- The APK also includes native/runtime libraries under
  `app/src/main/jniLibs/` and `app/src/main/assets/openclaw/native-libs/`,
  including Node/runtime dependencies, OpenSSL, ICU, sqlite, zlib, c-ares, and
  libc++.

Before publishing release binaries, complete this directory with the exact
license texts and notices required by those bundled binary components.
